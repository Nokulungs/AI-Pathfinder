# AI-Pathfinder
AI Fundamentals Knowledge Assistant  An interactive educational chatbot designed to simplify complex AI concepts. Built as a practical application of AI literacy, this assistant handles natural language queries regarding Machine Learning, NLP, and Deep Learning, providing structured learning paths and curated resources for aspiring AI students.
Project Title: AI Concept Chatbot

Description: A specialized conversational agent developed to bridge the gap between complex AI terminology and beginner-level understanding. This project was built to synthesize learning from IBM, Google Cloud, and DeepLearning.ai certifications into a functional, user-facing tool.

Key Features:

Knowledge Base: Trained on 15+ curated Q&A pairs covering LLMs, Neural Networks, and Computer Vision.

Structured Learning Flows: Includes guided paths for "AI Ethics" and "NLP Fundamentals."

Contextual Intelligence: Supports multi-turn conversations with up to 3 follow-up questions.

Resource Integration: Dynamically recommends "Further Learning" modules and provides citations from industry-leading courses.

Tech Stack:Landbot OpenAI API, NLP, Logic Branching.
